# PulseFabric Monitor

> GitHub Codespaces + Copilot Demo

Author — Mingchieh Wang, Wei Tien Pang
Date: March 23, 2026

This project rearchitects the B2B Design-to-Code workflow, eliminating complex local environments and high-risk manual handoffs. Our 'AI Design Cloud Sandbox' securely syncs UI, code, and AI editing. By enforcing read-only reviews and strict boundaries, the system absorbs environmental risks, enabling safe, zero-friction PRs.

---

## Getting Started with Codespaces

1. Click the green **Code** button on the repository page.
2. Select the **Codespaces** tab.
3. Click **Create codespace on main**.
4. Wait for the Dev Container to build — Node.js 20, Angular CLI, and all VS Code extensions will be installed automatically.
5. Once the terminal is ready, you're good to go.
